# Little Lemon Menu App

A SwiftUI menu app created as a practice project for building reusable views, working with mock data, navigation, filtering UI, and basic unit testing.

## Overview

This app displays a restaurant menu with three main categories:

- Food
- Drinks
- Desserts

Each menu item includes a title, price, category, order count, and ingredients. Users can browse the menu, open a detail screen for each item, and view additional information about the selected menu item.

## Features

- SwiftUI user interface
- Menu grouped by category
- Reusable menu item components
- Menu item detail screen
- Navigation using `NavigationStack` and `NavigationLink`
- Mock data for food, drinks, and desserts
- Bottom sheet filter screen
- Custom menu item protocol
- Basic unit test for menu item title initialization
